Challenge 4: Lists, Lists, and More Lists

Numbers, Bullets, and Definitions

There are three types of lists:

definition lists
which show terms and definitions
unordered lists
which show bullets
ordered lists
which show numbers or letters

Above is an example of a definition list.

Unordered lists can have a type attribute which can be applied to the UL or the LI tag. The three values allowed are:

  • type="square"
  • type="disc"
  • type="circle"

Ordered lists can have a type attribute, too. Used with the start attribute, they can show:

  1. lowercase letters
  2. uppercase letters
  3. lowercase Roman numerals
  4. uppercase Roman numberals
  5. numbers
